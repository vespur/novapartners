# 식스샵 프로 템플릿 지원금 조건 (200만원 기준)

> 식스샵 프로 템플릿 심사 기준 + 블록 개발 가이드 통합 정리

---

## 🎯 템플릿 지원금 등급

| 등급 | 금액 | 조건 |
|------|------|------|
| **기본 지원금** | 50만원 | 커스텀 블록 5개 이상 |
| **추가 지원금** | 1~200만원 | 커스텀 블록 10개 이상 + 새로 제작한 블록 5개 이상 |

**목표: 200만원 달성**
- ✅ 커스텀 블록 10개 이상
- ✅ 새로 제작한 블록 5개 이상
- ✅ 내 페이지 3개 이상 (필수)

---

## 📋 필수 요건 (1항목 미충족 = 즉시 반려)

### 1-1) 편집 용이성

- [ ]  에디터 내 모든 기능 오류 없이 작동
  - 커스텀 블록이 정상 동작하는가?
  - 설정값 변경 시 즉시 반영되는가?

- [ ]  사용자 편집 가능한 섹션/블록만 포함
  - ❌ 통이미지 사용 금지
  - ❌ 코드로만 만들어진 편집 불가능한 요소 금지
  - ✅ 텍스트, 이미지, 스타일 모두 에디터에서 편집 가능

- [ ]  콘텐츠 및 리소스 저작권 문제 없음
  - 이미지, 동영상, 아이콘 모두 재사용 가능 라이선스
  - (문제 발생 시 식스샵 책임 없음)

### 1-2) 콘텐츠 구성

- [ ]  **내 페이지 3개 이상** (필수)
  - 각 페이지의 레이아웃/내용물이 서로 다른가?
  - 기본 제공 페이지(상품 목록/게시판)와 함께 온전한 헤더/푸터 메뉴 구성

- [ ]  동일한 페이지 구성 반복 없음
  - 같은 디자인의 페이지가 여러 개 있으면 안 됨

- [ ]  **단순 상품 진열 위주 아님** (권장: 홈페이지 템플릿 또는 홈+상품판매)
  - 예: https://bricksand.co.kr/ 같은 구조

- [ ]  고품질 이미지만 사용 (저품질 이미지 = 반려)
  - ❌ 지양: 낮은 해상도, 뿌옇고 노이즈 낀 이미지
  - ❌ 지양: 과한 그림자/그라데이션/윤곽선 (구식 스타일)
  - ❌ 지양: 가짜 오피스, 웃는 비즈니스맨 등 스톡이미지 티
  - ❌ 지양: 과한 보정, 네온톤, 과한 대비
  - ✅ 추천: 자연광 기반 선명한 촬영
  - ✅ 추천: 실제 사용 맥락이 느껴지는 라이프스타일
  - ✅ 추천: 최신 트렌드 톤앤매너

- [ ]  이미지/동영상/아이콘/구성 요소 디자인 일관성
  - 전체 페이지에서 톤앤매너가 일치하는가?

- [ ]  의미 없는 더미 텍스트 없음
  - 실제 콘텐츠 또는 의미 있는 예시 텍스트만 사용

## 🌟 추가 요건 (지원금 관련, 미달성 가능)

### 2-1) 기본 지원금 (50만원) - 필수

- [ ]  **커스텀 블록 5개 이상** 포함
  - 마켓플레이스 블록 + 변형한 블록 + 새로 만든 블록 모두 포함

### 2-2) 추가 지원금 (1~200만원) - 목표

- [ ]  **커스텀 블록 10개 이상** 포함
- [ ]  **이 중 5개 이상이 새롭게 제작한 블록**
  - 바닥부터 새로 만든 블록 OR
  - 마켓플레이스 블록을 유의미하게 변형 (원본과 완전히 차별화)
- [ ]  **내 페이지 5개 이상** (선택, 권장)
- [ ]  **커스텀 섹션 + 그룹 블록**으로 차별화된 레이아웃 구성
- [ ]  **테마 설정 > 커스텀 코드**로 디자인 변형 (편집 용이성 해치지 않을 범위)

### 품질 심사 기준

- [ ]  차별화된 기능

  - 로딩/스크롤 시
  - hover/클릭 시
  - 슬라이드 시 등
- [ ]  차별화된 레이아웃
- [ ]  차별화된 디자인 요소 (아이콘, 버튼, 인디케이터 등)

## 🔍 커스텀 블록 점검표 (3항 - 새롭게 제작한 블록의 경우)

### 3-1) 설정 패널 정리

- [ ]  블록 이름: 한글 작성 ✅
- [ ]  모든 문구: 한글 작성 ✅ (기술 용어, 고유명사는 영어 가능)
- [ ]  TITLE로 관련 설정 그룹화
  - 복잡한 패널은 TITLE로 섹션 분리
  - 2번째 TITLE부터 자동 구분선 생성
- [ ]  외부 API 필요 시: 연동 방법 충분한 안내
  - 별도 문서 필요 시 DESCRIPTION에 링크 추가
- [ ]  isVisible 속성: 적절히 문법 오류 없이 사용
  - LIST 내부에서는 [index] 사용
  - 비교 연산자 가능
- [ ]  글자 요소: 글자 크기 설정 필수 (예외 가능)

### 3-2) 블록 동작 및 코드

- [ ]  너비: 커스텀 섹션 너비 초과 안 함
  - position: relative 블록 필수
  - (absolute, fixed, sticky는 예외 가능)

- [ ]  설정값 즉시 반영: bm.onContextChange 구현
  ```javascript
  function init() {
    // 초기화 로직
  }
  init();
  bm.onContextChange = () => {
    init();
  };
  ```

- [ ]  데스크톱 + 모바일 양쪽 정상 작동

- [ ]  동일 블록 여러 개: 독립적 작동
  - 1개 블록 수정 시 다른 블록 영향 없음

- [ ]  필요한 라이브러리: 명시 및 제공
  - 공통 코드 편집에 추가 시 명시

- [ ]  라이브러리 영향 범위: 다른 섹션/블록 영향 없음
  - 특히 CSS 격리 필수

- [ ]  bm.onDestroy 필요 시 구현
  ```javascript
  bm.onDestroy = () => {
    window.removeEventListener('scroll', handleScroll);
    // 정리 작업
  };
  ```

- [ ]  console.log 모두 삭제 (필수)
  - 최종 코드에 디버깅 문구 남으면 안 됨

### 3-3) 테마 설정 상속

- [ ]  제목글: font-family, font-weight 적용
  ```css
  font-weight: var(--font-weight-heading);
  font-family: var(--font-family-heading);
  ```

- [ ]  줄글: font-family, font-weight 적용
  ```css
  font-weight: var(--font-weight-body);
  font-family: var(--font-family-body);
  ```

- [ ]  색상 구성 (COLOR_SCHEME): CSS 변수 적용
  ```css
  background-color: var(--color-background-100);
  color: var(--color-text-100);
  border-color: var(--color-border-100);
  ```

### 3-4) 개별 설정

- [ ]  RICH_TEXT: 굵게, 기울임꼴 작동 ✅
- [ ]  RICH_TEXT: HTML 정상 출력 (3중 괄호 {{{property.id}}} 사용)
- [ ]  RICH_TEXT: 필요 없는 속성 숨김 (exclude 사용)
- [ ]  RANGE: unit(단위) + step(조절 단위) 적절히 적용
- [ ]  LINK: 새 탭 열기 함께 제공

### 3-5) 헤더 블록 별도 점검 (해당 시)

- [ ]  빈 링크 클릭 방지 (필수)
  ```javascript
  bm.container.addEventListener('click', (e) => {
    const link = e.target.closest('a');
    if (link) {
      const href = link.getAttribute('href');
      if (!href || href === '' || href === '#') {
        e.preventDefault();
        e.stopPropagation();
      }
    }
  }, true);
  ```

- [ ]  글자 크기 조정 기능 (모든 글자)
- [ ]  컨테이너 최대 너비 조절 (데스크톱)
  - max-width on/off 가능
  - max-width 값 조절 가능 (RANGE)
  - 좌우 padding 조절 가능

### 3-6) 푸터 블록 별도 점검 (해당 시)

- [ ]  빈 링크 클릭 방지 (필수)
- [ ]  국내 쇼핑몰 푸터: 법적 필수 정보 포함
  - 상호명
  - 대표자명
  - 사업자등록번호
  - 통신판매업 신고번호
  - 주소
  - 대표 전화번호
  - 이메일 주소

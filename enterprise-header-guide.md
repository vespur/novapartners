# 엔터프라이즈 글로벌 헤더 블록 - 설치 가이드

## 📦 파일 구성

이 패키지는 2개의 파일로 구성되어 있습니다:

- **`enterprise-header-block.html`** - 블록 코드 (복사하여 코드 편집 영역에 붙여넣기)
- **`enterprise-header-settings.json`** - 설정 스키마 (복사하여 세팅 빌더에 붙여넣기)

---

## ✨ 주요 기능

### 🎯 프리미엄 기능
- **3단계 네비게이션**: 메인 메뉴 → 드롭다운 → 메가 메뉴
- **메가 메뉴**: 3개 카테고리로 구성된 넓은 드롭다운
- **완벽한 반응형**: PC/태블릿/모바일 자동 대응
- **모바일 햄버거 메뉴**: 슬라이드 네비게이션
- **유틸리티 링크**: 최대 3개 추가 링크
- **다국어 지원**: 언어 선택기 내장

### 🎨 비개발자 친화적
- **40개 이상의 설정 옵션**: 코드 수정 없이 모든 것을 에디터에서 조절
- **카테고리별 구성**: 로고, 디자인, 메뉴, 모바일, 고급 설정
- **친절한 설명**: 모든 옵션에 상세한 안내 포함
- **실시간 미리보기**: 변경 즉시 확인

## 🚀 설치 방법 (5단계)

### 1단계: 블록 메이커 접속

1. **식스샵 프로 대시보드** 로그인
2. 좌측 메뉴에서 **"디자인"** 클릭
3. **"블록 메이커"** 선택
4. 우측 상단의 **"+ 새 블록 만들기"** 버튼 클릭

---

### 2단계: 블록 기본 정보 입력

새 블록 생성 화면에서 다음 정보를 입력하세요:

```
블록 이름: 엔터프라이즈 글로벌 헤더
카테고리: 헤더
설명: 3단계 네비게이션과 메가 메뉴를 지원하는 프리미엄 헤더
```

**"만들기"** 버튼을 클릭하여 블록을 생성합니다.

---

### 3단계: 블록 코드 붙여넣기

1. 블록 메이커 편집 화면에서 **"코드"** 또는 **"코드 편집"** 탭 선택

2. `enterprise-header-block.html` 파일을 열어서 **전체 내용을 복사**

3. 코드 편집 영역에 **붙여넣기** (Ctrl+V 또는 Cmd+V)

4. 코드가 다음 구조로 되어 있는지 확인:
   ```html
   <style>
     /* CSS 코드 */
   </style>

   <template>
     <!-- HTML 마크업 -->
   </template>

   <script>
     // JavaScript 코드
   </script>
   ```

---

### 4단계: 설정 스키마 가져오기

1. 블록 메이커 편집 화면에서 **"세팅"** 또는 **"설정"** 탭 선택

2. 우측 상단의 **⋮ (더보기 메뉴)** 클릭

3. **"JSON 가져오기"** 또는 **"Import JSON"** 선택

4. `enterprise-header-settings.json` 파일을 열어서 **전체 내용을 복사**

5. 나타난 입력창에 **붙여넣기**

6. **"가져오기"** 또는 **"확인"** 버튼 클릭

> **참고**: 식스샵 프로 버전에 따라 인터페이스가 다를 수 있습니다. JSON 가져오기 기능이 없는 경우, 고객센터에 문의하세요.

---

### 5단계: 저장 및 게시

1. 우측 상단의 **"저장"** 버튼 클릭

2. **"게시"** 또는 **"공개"** 버튼 클릭

3. 블록이 블록 라이브러리에 추가되었습니다! ✅

---

## 📝 페이지에 블록 추가하기

### 방법 1: 신규 페이지에 추가

1. **"페이지"** 메뉴로 이동

2. 원하는 페이지 선택 (또는 새 페이지 만들기)

3. 페이지 에디터에서 좌측 패널의 **"블록 추가"** 클릭

4. **"내 블록"** 또는 **"커스텀 블록"** 카테고리 선택

5. **"엔터프라이즈 글로벌 헤더"** 블록을 페이지 최상단으로 드래그

### 방법 2: 기존 헤더 교체

1. 기존 헤더 블록 선택 → 삭제

2. 위의 방법 1을 따라 새 헤더 추가

## ⚙️ 설정 방법

블록을 페이지에 추가한 후, 블록을 선택하면 **우측 설정 패널**이 나타납니다.

### 📸 로고 설정

#### 로고 이미지 업로드
1. **"로고 이미지"** 필드의 **"이미지 선택"** 클릭
2. 미디어 라이브러리에서 로고 선택 또는 새로 업로드
3. **권장**: SVG 형식 또는 투명 배경 PNG

#### 로고 크기 조정
- **로고 높이** 슬라이더를 드래그하여 조절
- **일반**: 24~40px
- **큰 로고**: 40~60px

#### 로고 링크
- **기본값**: `/` (홈페이지)
- 원하는 페이지 주소로 변경 가능

## 💡 사용 팁

### 메뉴 구조 설계

1. **메뉴는 4개 이하로** 유지하세요
   - 너무 많은 메뉴는 사용자 경험을 해칩니다

2. **메가 메뉴는 1개만** 사용하세요
   - 보통 "제품" 또는 "서비스" 메뉴에 적용

3. **하위 메뉴는 5~8개**가 적당합니다
   - 너무 많으면 스크롤이 필요해집니다

### 색상 선택

1. **대비 확인**: 텍스트와 배경의 대비가 충분한지 확인
   - [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) 활용

2. **브랜드 컬러**: 호버 색상에 브랜드 포인트 컬러 사용

3. **일관성**: 드롭다운 배경은 헤더 배경과 같거나 한 톤 다르게

### 모바일 최적화

1. **간결하게**: 모바일에서는 메뉴가 더 간결해야 합니다

2. **터치 영역**: 모바일에서 터치하기 쉽도록 충분한 간격 유지

3. **테스트**: 실제 모바일 기기에서 테스트하세요

## ❓ 문제 해결

### 블록이 표시되지 않아요

**원인**: 블록이 제대로 저장되지 않음

**해결**:
1. 블록 메이커에서 블록이 **"게시"** 상태인지 확인
2. 페이지 에디터를 새로고침 (F5)
3. 브라우저 캐시 삭제 후 재시도

---

### 메뉴가 표시되지 않아요

**원인**: 메뉴 설정값이 비어있음

**해결**:
1. 블록 선택 → 우측 설정 패널 확인
2. "메뉴 1 제목" 등 필수 필드 입력
3. "하위 메뉴 목록"에 올바른 형식으로 입력:
   ```
   메뉴명|/링크주소
   ```
4. 파이프 기호(`|`)가 제대로 입력되었는지 확인

---

### 드롭다운이 작동하지 않아요

**원인**: JavaScript 오류 또는 설정 문제

**해결**:
1. 브라우저 개발자 도구(F12) → Console 탭 확인
2. 에러 메시지가 있는지 확인
3. 블록 코드가 완전히 복사되었는지 확인 (`<script>` 태그 포함)
4. 다른 블록과 충돌 가능성 확인

---

### 모바일에서 메뉴가 이상해요

**원인**: 모바일 브레이크포인트 설정 문제

**해결**:
1. 설정 패널 → "모바일 설정" 확인
2. "모바일 전환 기준" 값 조정 (일반적으로 768px)
3. 실제 모바일 기기 또는 브라우저 개발자 도구의 모바일 뷰로 테스트

---

### 색상이 적용되지 않아요

**원인**: 색상 코드 형식 문제

**해결**:
1. 색상은 `#RRGGBBAA` 형식 (8자리 HEX)
2. 색상 선택기를 사용하여 선택
3. 직접 입력 시 `#FFFFFFFF` (완전 불투명 흰색) 형식 사용

---

### 로고가 너무 크거나 작아요

**해결**:
1. **PC**: "로고 높이" 슬라이더 조정 (20~60px)
2. **모바일**: "모바일 로고 높이" 슬라이더 조정 (16~40px)
3. 헤더 높이와의 균형 고려 (헤더 높이의 40~50%)

---

### 메가 메뉴가 화면을 벗어나요

**원인**: 카테고리 항목이 너무 많음

**해결**:
1. 각 카테고리당 5~8개 항목으로 제한
2. 또는 CSS 커스터마이징:
   ```css
   .dropdown.mega {
     max-width: 90vw;
   }
   ```

## 🔧 고급 커스터마이징

### Custom CSS 추가

에디터 패널에서 제공하지 않는 스타일을 추가하려면:

1. 블록 선택
2. 우측 패널에서 "고급" 또는 "CSS 클래스" 찾기
3. 커스텀 클래스 추가 (예: `custom-header`)
4. 페이지 설정 → 코드 삽입 → `<head>`에 추가:

```html
<style>
.custom-header .nav-link {
  text-transform: uppercase;
  letter-spacing: 1px;
}

.custom-header .dropdown {
  border-top: 3px solid #0066CC;
}
</style>
```

### JavaScript 이벤트 추가

메뉴 클릭 추적 등을 위한 커스텀 스크립트:

```html
<script>
document.addEventListener('DOMContentLoaded', function() {
  const header = document.querySelector('.global-header');

  header.addEventListener('click', function(e) {
    if (e.target.matches('.nav-link')) {
      // Google Analytics 등 추적 코드
      console.log('메뉴 클릭:', e.target.textContent);
    }
  });
});
</script>
```
